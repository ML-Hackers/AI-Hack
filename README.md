## Evaluation of LLMs:
- Metrics:
  - [Bleu Score](https://en.wikipedia.org/wiki/BLEU)
  - [BLEURT Score](https://github.com/google-research/bleurt)
  - Semantic Similarity
  - F-score / Precision / Recall (classification use cases)
